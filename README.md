# Web-Scrapping-Government-Jobs

A Python-based web scraping project that collects government job-posting data and organizes it for analysis

## Features

- Collects job-posting information from government of canada job bank
- Cleans and organizes the data
- Displays the results in a structured format

## Tools Used

- Python
- Jupyter Notebook
- Beautiful Soup (BeautifulSoup4)
- Pandas

## Instructions

1. Download or clone this repository.
2. Install the required Python libraries.
3. Open `Job-Postings-Web-Scraping-Tool.ipynb` in Jupyter Notebook.
4. Run the cells in order.
